## 🌐 What Is CloudTrip?

CloudTrip is a modern travel platform built for people who love smooth, organized, and inspiring travel.
Whether you’re planning a weekend escape or a full itinerary, CloudTrip brings everything into one intelligent hub:

* Discover curated destinations
* Book flights
* Keep all reservations synced in **My Bookings**
* Pay quickly and securely
* Enjoy a personalized travel experience

## ⚙️ Getting Started (Local Setup)

### Backend Setup

```bash
cd backend
python -m venv venv
venv/Scripts/activate   # Windows
# or
source venv/bin/activate   # macOS/Linux

pip install -r requirements.txt
python app.py
```
